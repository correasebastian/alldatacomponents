#alldata components

## requires
* nodejs >= 8.12.0

## install dependencies
```js 
npm install
```

## run in dev mode
```js
npm run startdev
```

# how to create a component



## install yeoman globally
```js
 npm install -g yeoman
```

## open the folder
```js
 cd generator-fractal-component/
```

## run the command
```js
 npm link
```

## go back to the root folder and run
```js
 yo fractal-component
```
